## Composition

This starter kit is composed of:

- [Node.js](https://nodejs.org/en)

  <br/>
- [Axios](https://axios-http.com/docs/intro)

- [Cheerio](https://cheerio.js.org/)

- [CSV Writter](https://www.npmjs.com/package/csv-writer)
